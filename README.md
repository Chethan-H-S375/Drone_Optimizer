🚁 Drone Delivery Optimization System

This project is an end-to-end drone delivery optimization system that combines machine learning and route optimization algorithms to improve delivery efficiency and success rates.

It simulates real-world drone logistics by predicting delivery success probabilities and computing optimal delivery routes using the Travelling Salesman Problem (TSP) approach.

🔧 Key Features
📊 Machine Learning Model
Predicts the probability of successful drone deliveries
Trained on historical delivery data
Helps filter or prioritize delivery routes
🗺️ Route Optimization
Implements TSP-based algorithms to find the most efficient delivery path
Supports both:
Optimization with ML predictions
Optimization without ML (baseline comparison)
⚙️ Backend API
Built with Python (likely FastAPI/Flask)
Serves prediction and routing endpoints
Includes testing support
🖥️ Frontend Interface
Simple UI to interact with the system
Allows users to input coordinates and visualize results
📁 Modular ML Pipeline
Data preprocessing and training scripts
Batch prediction support
